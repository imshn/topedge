  SUPPORT & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

If pages don't appear in Google Search Console after 2 weeks:
  ☐ Check that robots.txt allows Googlebot (should be default)
  ☐ Verify sitemap.xml is accessible at the root
  ☐ Look for crawl errors in GSC > Coverage
  ☐ Check server logs for 404 errors

If GA4 is not tracking:
  ☐ Open browser DevTools > Network tab
  ☐ Reload page and look for a request to "collect" from
    www.google-analytics.com
  ☐ If missing, verify GA4 ID is correct (not XXXXXXXXXX)
  ☐ Wait 24 hours for data to appear in GA4 dashboard

If schema validation fails:
  ☐ Use https://schema.org/validator to test each page
  ☐ Most common issue: missing closing tags (check file wasn't corrupted
    during upload)
  ☐ Re-upload the HTML file and clear browser cache
